SO(2)-equivariant kernels cannot exploit per-vertex mesh-frame orientation, which varies arbitrarily across the surface. This forces the network to learn frame-independent, physics-driven features for Cp and WSS — acting as an intrinsic geometric regularizer. No such constraint applies to the Cd head (global mean-pool → MLP), which memorises drag on ~160 training cars.
